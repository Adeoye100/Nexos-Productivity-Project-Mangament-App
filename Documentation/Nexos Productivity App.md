# Nexos Productivity App

## Overview
Nexos is an all‑in‑one productivity dashboard that combines task management, habit tracking, AI assistance, weather insights, customizable backgrounds, and a searchable command library into a single, keyboard‑friendly interface.

## Features

- [[Task Manager]] – create, prioritize, schedule, and track tasks with lists, kanban board, due dates, reminders, AI suggestions, and Vim‑style navigation.
- [[Habit Tracker]] – visual yearly heatmap, streaks, habit categories, frequencies, edit‑past mode, Vim navigation.
- [[GitHub Integration]] – visualize your coding contributions with a GitHub activity heatmap and recent commit list.
- [[AI Assistant]] – chat‑based LLM that can read your tasks, create/complete tasks, give weather insights, and provide productivity tips.
- [[Command Manager]] – searchable repository of CLI commands (Git, SSH, Linux, etc.) with custom command support, copy‑to‑clipboard, tagging.
- [[Background Manager]] – set custom images or default backgrounds, adjust opacity, blur, black‑&‑white toggle, parallax scrolling.
- [[Notification Log]] – view and manage task‑due, overdue, completed, AI reply, and reminder notifications.
- [[Settings Panel]] – configure user profile, location, notification preferences, reset data.
- [[Weather Dashboard]] – current weather (temperature, condition, humidity, wind, visibility, sunrise/sunset, feels like, pressure), unit toggle, location search, weekly forecast, AI‑driven suggestion.
- [[Weekly Forecast]] – 7‑day outlook with daily summaries.
- [[Onboarding Flow]] – guided introduction for new users.
- [[Keyboard Shortcuts]] – cheat sheet of Vim‑style navigation and command shortcuts.
- [[Theme & Appearance]] – light/dark theme toggle, customizable accent colors.
