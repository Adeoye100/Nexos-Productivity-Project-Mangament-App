# GitHub Integration

## Overview
Nexos integrates with GitHub to display your coding activity directly alongside your habits. This provides a comprehensive view of both your personal habits and professional/coding contributions.

## Features
- **Contribution Heatmap** – A GitHub-style yearly contribution graph showing commit frequency.
- **Recent Activity** – A list of the most recent commits from your specified repository.
- **Secure Integration** – Connect using a Personal Access Token (PAT) which is encrypted before being stored locally.
- **Sync Control** – Manually refresh activity to get the latest commit data.

## Configuration
To enable GitHub integration:
1. Go to the [[Settings Panel]].
2. Enter your **GitHub Personal Access Token** (requires `repo` scope for private repositories).
3. Enter your **Repository Path** (in the format `username/repo-name`).
4. Click **Test Connection** to verify settings.
5. Save your settings.

## Privacy & Security
- Your GitHub token is **never** sent to any Nexos server.
- The token is encrypted using AES-256 before being saved to your browser's `localStorage`.
- All requests to the GitHub API are made directly from your browser.
